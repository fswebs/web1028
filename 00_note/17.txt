포토샵
### select(선택) 메뉴
All -> 전체 선택 ctrl+a
Deselect -> 선택 해제, ctrl+d
inverse -> 선택 반전, ctrl+shift+i

Modify(수정) -> 왜곡되는 현상이 있다.
  Border -> 테두리
  Smooth -> 부드럽게
  Expand -> 확장
  Contract -> 축소
  Feather -> 선택 후 옵션 지정

[비교] 선택 옵션의 feather(페더) -> 옵션 지정 후 선택
Vignetting(비네팅) 효과
사진의 외곽 부분이 어두워지는 그늘 현상. 전문적으로는 주변부 광량 저하 현상

Transform Selection -> 선택 영역 변형

Load Selection 
  레이어의 썸네일을 ctrl+클릭 -> 레이어의 색상 부분이 선택된다.
